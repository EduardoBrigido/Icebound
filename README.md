# Icebound
Mod para Minecraft 1.20.1 (Forge) que adiciona um bioma, com novos itens, mobs, mecânicas de frio e muito mais. Disponível apenas no idioma inglês
## Requisitos
- Minecraft 1.20.1
- Forge 47.4.0
- TerraBlender 3.0.0.169

## Funcionalidades

### Bioma
- Snowy Beach: bioma de praia nevada com areia congelada e palmeiras 

### Itens e Blocos
- Crystallum: minério encontrado em biomas frios
- Armadura de lã e couro: protege o jogador do frio
- Espada de Crystallum, Enxada de Crystallum
### Alimentos
- Morango, Iceberry, Sorvete, Chocolate Quente

### Mobs
- Penguin: Mob agressivo
- Fire Raccoon: Mob amigável domesticável

### Mecânicas
- Sistema de frio
- Frio causa lentidão, fraqueza e dano gradual
- Armaduras reduzem o resfriamento

## Instalação
1. Instale o Forge 1.20.1
2. Instale o TerraBlender 3.0.0.169
3. Coloque o arquivo `.jar` na pasta `mods`
